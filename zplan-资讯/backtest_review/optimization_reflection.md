# 选股系统优化反思与方向

> 2026-06-11，基于 v1 vs v2 完整对比实验的总结

## 做了什么

| 实验 | 方法 | 结论 |
|------|------|------|
| 规则分审计 | 14 日期/5 万条 stock_rule_scores 回测 | Spearman ρ < 0.03，TOP300 超额为负 |
| v1 vs v2 因子回测 | 98 日期/70 万条，11 种评分方法 | 所有方法 ρ < 0.02，差异在噪音范围 |
| v1 vs v2 真实管道模拟 | 4 个历史周五，跑 init-rule + TOP300 | **v2 全部跑输，5 日超额 -1.21%** |
| 处罚权重校准 | 350 条有前向收益的 LLM 评估记录 | 旧系统数据噪音大，仅基本面恶化有效 |
| 管道唯一化 | morning_top10 + evening_review | ✅ 正确且必要 |
| 处罚参数化 | strategy.yaml + strategy.py + llm_research.py | ✅ 好的基础设施 |
| C3 硬截断 | ret_20d > 5% 不入 LLM | ✅ 合理兜底 |

## 核心发现

### 1. 因子优化是死胡同

```
所有 11 种评分方法的 20 日 Spearman ρ < 0.03
换因子、调权重、加概念、加筹码峰——差异不超过 0.01
ρ = 0.02 的"最优"方法，本质上还是随机
```

**启示**：不要花时间在因子层面做微调。选 v1 还是 v2、调 +3 还是 +5 分，都不改变 ρ 天花板。

### 2. "合理性"不等于"有效性"

v2 的反转因子逻辑上无懈可击——超卖应回归均值。但 2026 年 5 月 A 股是单边下跌市，超卖股继续跌。我们基于信念（"追涨不好，低吸好"）做了优化，但数据证明信念是错的。

**启示**：每个优化必须有历史数据支撑，不能凭"合理"就改。合理只是假设，回测才是证据。

### 3. LLM 是唯一有区分度的信号

| 组别 | 样本 | 均收益 | 均规则分 |
|------|------|--------|---------|
| Pass | 13 | **+6.11%** | 65.6 |
| Fail | 337 | +0.25% | 53.5 |

LLM 能把 pass 率从 3.7% 提到有意义的方向性预测。规则分差异只有 12 分（65 vs 53），但收益差异是 5.86%。

**启示**：α 来自 LLM 的判断力，不来自因子。优化资源应 80% 投入 LLM 层，20% 投入因子层。

### 4. 市场状态切换因子有效性

| 年度 | 5 日 ρ |
|------|--------|
| 2024 | +0.061 |
| 2025 | +0.072 |
| **2026** | **-0.002** |

因子在 2024-2025 还有微弱正信号，2026 年直接归零甚至变负。2026 年是 AI/机器人主题驱动市，传统量价因子全面失效。

## 优化方向重定义

### 不再做的事

- ❌ 纠结 v1 vs v2 vs v3 — ρ 天花板太低，换什么都没本质区别
- ❌ 基于"合理性"调参数 — 必须先有数据支撑
- ❌ 在高噪音数据上跑校准 — 等干净数据积累

### 应该做的事

**A. LLM 信号增强（高优先级）**
- prompt 中注入更多结构化数据（板块排名、资金流向、概念热度变化）
- 让 LLM 做「这个票明天会涨还是跌」的二分类判断，而非模糊的"关注/观望"
- 积累 LLM 判断 vs 实际结果的对照数据，校准 LLM 的置信度

**B. 市场状态自适应（中优先级）**
- 检测当前市场状态（趋势/震荡/下跌/主题）
- 不同状态用不同的因子配置或直接调整候选池大小
- 下跌市中 TOP300 → TOP100（缩小范围、提高质量）

**C. 等数据，再校准（进行中）**
- 新系统（v1 + 买价修复 + LLM 投资分析师）跑 2-4 周
- 积累 > 500 条有前向收益的干净评估数据
- 用 `penalty_calibrate.py` 产出可信的处罚权重建议

**D. 每次改动必须先过历史管道模拟**
- `simulate_historical_picks.py` 在 4+ 个历史日期上跑
- 产出 v_old vs v_new 对比报告
- 新方案在所有日期/周期均不差于旧方案 → 才合并

## 下次优化应该怎么做

```
1. 提出假设："改 X 可以提升前向收益"
2. simulate_historical_picks.py 在 4 个历史日期上验证
3. 如果 3/4 日期跑赢 → 合并
4. 如果跑输 → 放弃假设，不做
5. 积累 10 次实验 → 统计哪些假设更可能正确
```

**核心原则**：不做没有历史数据支撑的优化。信念是廉价的，回测数据是唯一可以信任的。
